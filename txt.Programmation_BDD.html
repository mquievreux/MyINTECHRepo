    • Bonnes pratiques
      • ID 0 anonyme "bonne pratique" utilisé par monsieur spinelli pour éviter l'utilisations de la valeur null. l'ID 0 d'une entitée est initialisé a ' ' (vide), ce tuple est utilisé lorsque, par exemple, une formation n'a pas d'auditeur. on l'associ a l'auditeur 0 (ce qui signifie) pas d'auditeur du tout, au lieu de passer NULL (vade retro satanas !)
        • généralisable a toute les entités
        • ne pas utiliser sur des tables de jointures

      • Constraintes

          • constrainte interne CREATE TABLE Persons(P_Id int NOT NULL,LastName varchar(255) NOT NULL,FirstName varchar(255),Address varchar(255),City varchar(255),CONSTRAINT chk_Person CHECK (P_Id>0 AND City='Sandnes'))
          • contrainte externe CREATE TABLE Persons(P_Id int NOT NULL,LastName varchar(255) NOT NULL,FirstName varchar(255),Address varchar(255),City varchar(255),CONSTRAINT chk_Person CHECK (P_Id>0 AND City='Sandnes')) ALTER TABLE Persons Add constrainte UK_Person_Address unique (Address); //attention, syntaxe approximative//
